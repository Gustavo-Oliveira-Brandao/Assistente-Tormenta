<h1>Assistente T20</h1>

Esse projeto almeja prover uma experiência digital intuitiva e automatizada de gerenciar fichas de Tormenta20.

Para executar o projeto basta instalar a dependências com "npm install" e executar com "npm run dev"

Tormenta 20 pertence a Jambo Editora. Todos os direitos são reservados a editora.
